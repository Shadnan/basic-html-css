What is inline elements?

Inline elements display in a line. They do not force the text after them to a new line. An anchor (or link) is an example of an inline element. Above tags are also inline elements

What is block elements

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can). Two commonly used block elements are "P" tag and "div" tag.

What is lnline-block?

Inline-Block Elements: These elements are displayed as inline elements, meaning they do not start on a new line, but they can have set widths and heights like block elements. This allows them to flow within text or other inline elements while also having block-level properties. Using the display: inline-block property allows elements to behave like inline elements in terms of their placement within text or other inline elements while still allowing for the application of properties like width, height, margins, and padding as you would with block-level elements. For example, you might use inline-block for creating navigation menus where each menu item is a block with its own width and height but flows inline with other menu items.
